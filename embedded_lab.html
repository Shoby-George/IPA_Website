DSP Based Experiments

The aim of the experiment is to control the speed of the 3 phase induction motor and brushless DC motor. Experiment is based on Texas Instruments TMS320LF2407A DSP processor which can be operated either in mode 1(serial mode) or in mode 2(standalone mode). User can interact with the trainer through IBM PC keyboard and 16 x 2 LCD display. One RS232C serial Interface using On-chip serial communication Interface( SCI ) module. It also uses an Intelligent Power module - PEC160SMO1


ABB IRB1200 Experiment

Here we are doing experiment on ABB IRB 1200 robot. This is an 6- Degree of freedom industrial Robot. This robot mainly use for pick and place application in industry. By using Teach pendent we can control the movement of various joints of robot.


Microcontroller Based Experiments

The aim is to control the speed of the stepper motor by using 8051 microcontroller. And also to generate Square wave and Saw tooth waveform by using the 80196 microcontroller.


Linear Induction Motor Control

The aim of the experiment is to study about the linear induction motor and perform traction test on it. In this experiment the speed of the motor can be varied by adjusting the frequency and modulation index value while the motor is in motion.

Eddy Current Drive

The Eddy Current Drive is used to control speed of the induction motor. It is a combination of constant speed AC induction motor, coupled to an eddy current clutch. The function of eddy current clutch is to transmit a controlled torque from induction motor to the load. The medium of torque transmission is a controlled magnetic field generated by a stationary field coil.
